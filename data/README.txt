AI OPERATIONS ASSISTANT - E-COMMERCE INVENTORY SYSTEM
======================================================

PROJECT OVERVIEW
----------------
Natural language query system for e-commerce inventory and product management.

EXAMPLE QUERIES
---------------
- "Fire TV Stick'ten kaç tane stokta var?"
- "En yüksek rating'li Electronics ürünü hangisi?"
- "Bu ürün yoksa alternatif ne önerebiliriz?"

FILES INCLUDED
--------------
1. final_product_database.csv - Main product database (1,200 products, 21 fields)
2. products_with_alternatives.csv - Products with alternative recommendations (23 fields)
3. products_for_ai.csv - AI-ready with natural language descriptions (22 fields)
4. dashboard_summary.csv - Executive dashboard statistics
5. data_schema.txt - Complete field documentation

DATASET STATISTICS
------------------
